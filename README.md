# Автоматизированные UI тесты

Этот проект реализует автоматизированные тесты для
[тестового приложения Practice Automation](https://practice-automation.com/). Тесты
написаны с использованием **Python**, **Pytest**, **Allure** и **Playwright**.

## Обзор проекта

Цель этого проекта — автоматизировать тестирование учебного сайта Practice Automation, который содержит различные
элементы для тренировки автоматизации: формы, модальные окна, попапы, слайдеры, выпадающие списки и другие
интерактивные компоненты. Автоматизированные тесты проверяют различные функции приложения, чтобы обеспечить его
стабильность и корректность.

Проект разработан с использованием современных подходов к автоматизации тестирования:
- **Page Object Model (POM)** для структурированного представления страниц
- **Компонентный подход** для переиспользуемых элементов интерфейса
- **Фикстуры Pytest** для управления тестовыми данными и окружением
- **Allure** для подробных отчетов о выполнении тестов
- **Playwright** для надежного взаимодействия с браузером
- **UI Coverage Tool** для отслеживания покрытия тестами элементов интерфейса

## Начало работы

### Клонирование репозитория

Чтобы начать, клонируйте репозиторий проекта с помощью Git:

```bash
git clone https://github.com/DaniNikolaev/practice-automation-autotests-ui.git
cd practice-automation-autotests-ui
```

### Создание виртуального окружения

Рекомендуется использовать виртуальное окружение для управления зависимостями проекта. Следуйте инструкциям для вашей
операционной системы:

#### Linux / MacOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Установка зависимостей

После активации виртуального окружения установите зависимости проекта, перечисленные в файле `requirements.txt`:

```bash
pip install -r requirements.txt
```

### Дополнительная настройка Playwright (при необходимости)

Если вы запускаете Playwright впервые, вам может потребоваться установить необходимые браузеры:

```bash
playwright install
```

### Запуск тестов с генерацией Allure-отчета

Чтобы запустить тесты и сгенерировать Allure-отчет, используйте следующую команду:

```bash
pytest -m "regression" --alluredir=./allure-results
```

Это выполнит все тесты в проекте и отобразит результаты в терминале.

### Просмотр Allure-отчета

После выполнения тестов вы можете сгенерировать и просмотреть Allure-отчет с помощью:

```bash
allure serve allure-results
```

Эта команда откроет Allure-отчет в вашем браузере по умолчанию.
